🔧 管理后台HTTP服务激活指引

⚠️ 重要： 管理后台需要激活云开发的HTTP服务才能正常工作。
HTTP服务是免费功能，有免费额度，激活后不会产生额外费用。

方法一：通过云开发控制台激活

1
打开云开发控制台
点击以下链接直接进入环境详情页：
打开云开发控制台 →
2
找到HTTP服务入口
云开发控制台 → 环境列表 → 点击环境【cloud1】 → 左侧菜单 → 【云函数】 → 页面顶部标签【HTTP服务】

如果没有找到，尝试：
左侧菜单最底部【更多】 → 【HTTP服务】
3
激活HTTP服务
在HTTP服务页面：
1. 点击【开通】或【启用】按钮
2. 选择【按量付费】（有免费额度）
3. 点击【确认】或【提交】
4
验证激活
激活后，点击下方按钮验证：

方法二：直接链接（如果方法一失败）

备用
尝试以下直接链接
HTTP服务管理页面 → 云函数HTTP服务 →

方法三：通过微信小程序控制台

如果你的云开发环境关联了微信小程序，也可以通过微信小程序控制台激活匿名登录作为替代方案。
微信
启用匿名登录
微信公众平台 → 【开发】 → 【开发管理】 → 【云开发】 → 选择环境 cloud1 → 【设置】 → 【安全设置】 → 【登录授权】 → 启用【匿名登录】
✅ HTTP服务已激活！
现在可以访问管理后台了：
前往管理后台 →
❌ HTTP服务尚未激活
请继续按照上述步骤操作，完成后刷新此页面重新检查。

遇到问题？

如果以上方法都找不到入口，请：

  1. 确认你登录的是腾讯云账号（不是微信账号）
  2. 确认账号有云开发环境的访问权限
  3. 尝试清除浏览器缓存后重新打开控制台
  4. 或联系技术支持
