## 端到端测试

### 用户反馈流程

1. 用户在小程序提交反馈
2. 管理员在后台看到新反馈
3. 管理员回复反馈
4. 用户在"我的反馈"看到红点提示
5. 用户查看详情看到回复

- [ ] 状态流转正确（pending -> replied）
- [ ] hasReply 状态正确
- [ ] 时间戳记录正确
- [ ] replies 数组正确更新
