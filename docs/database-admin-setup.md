# admin 数据库集合设置指南

## 操作步骤

1. 打开微信开发者工具
2. 点击顶部"云开发"按钮
3. 进入"数据库"标签
4. 点击"添加集合"，输入集合名称：`admin`
5. 点击"确定"

## 集合结构

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| _id | string | 是 | 自动生成 |
| username | string | 是 | 管理员用户名 |
| password | string | 是 | 密码（哈希后的值） |
| createdAt | string | 是 | 创建时间（ISO格式） |

## 初始化管理员账号

在云数据库控制台中插入第一条管理员记录：

```json
{
  "username": "admin",
  "password": "YWRtaW4xMjM0NTY=",
  "createdAt": "2026-05-24T00:00:00.000Z"
}
```

**说明**：
- 密码字段是 `admin123456` 的 base64 编码
- 实际生产环境应使用更强的哈希算法（如bcrypt）
- 登录云函数会验证 base64 解码后的值

## 安全提示

1. 生产环境请修改默认密码
2. 建议使用更强的密码哈希算法
3. 限制admin集合的访问权限
