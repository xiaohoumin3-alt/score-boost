# 产品功能演示视频脚本

## 视频概述
- **时长**: 60-90秒
- **风格**: 轻快、科技感、教育向
- **目标**: 展示AI原生功能，突出与传统刷题的区别

---

## 脚本

### 开场 (0-5秒)
```
画面：黑屏，白色文字逐字出现
"生地会考倒计时30天"
"还在盲目刷题？"

音效：倒计时滴答声
```

### 痛点呈现 (5-15秒)
```
画面1：学生埋头刷题，表情疲惫
字幕："100道题，70道会的浪费时间"

画面2：试卷上红叉，分数卡住
字幕："会的重复做，不会的还是不会"

画面3：家长焦虑表情
字幕："不知道孩子哪里薄弱？"
```

### 产品亮相 (15-25秒)
```
画面：手机屏幕特写，小程序界面展开
配音："日日守护，AI帮你精准复习"

画面切换：
- 首页卡片展示
- AI生成题目动画
- 自适应难度可视化

配音："不是题库抽题，而是AI根据你的薄弱点实时生成"
```

### 核心功能1：AI个性化出题 (25-35秒)
```
画面：分屏对比
左边：传统App - 题目从题库抽取
右边：日日守护 - AI实时生成

字幕："每个学生，题目都不同"

画面：AI思考动画 → 题目生成
配音："AI分析你的答题情况，实时生成专属题目"
```

### 核心功能2：自适应难度 (35-45秒)
```
画面：游戏化进度条
连续答对3题 → 难度下降（绿色）→ 文字"建立信心"
连续答错2题 → 难度上升（橙色）→ 文字"找准盲区"

配音："自动调整难度，不会太难也不会太简单"
```

### 核心功能3：智能提示 (45-55秒)
```
画面：学生答错题
弹窗出现智能提示（不是直接给答案）

三种提示类型动画：
- "检查符号"（粗心错误）
- "展示步骤"（计算错误）
- "讲解概念"（概念错误）

配音："不直接给答案，分级提示培养思考能力"
```

### 效果展示 (55-65秒)
```
画面：数据可视化动画
模拟考分数从65分 → 88分

字幕："30天，每天15分钟，精准提升"

画面：Memory系统界面
知识点掌握度地图

配音："Memory系统记住你的学习情况，自动生成复习计划"
```

### 行动号召 (65-75秒)
```
画面：二维码 + 小程序名称
"日日守护"

字幕："前100名免费体验7天"
"微信搜索：日日守护"

配音："生地会考只有一次，让AI成为你的专属家教"
```

### 结尾 (75-80秒)
```
画面：Logo + Slogan
"AI精准复习，轻松拿A+"

音效：轻松收尾音
```

## 拍摄/制作要点

### 屏幕录制
1. 使用真实手机屏幕录制
2. 操作速度适中，让观众看清
3. 重点功能要放大展示

### 动画效果
1. AI生成题目：使用"思考中→生成"动画
2. 自适应难度：使用进度条颜色变化
3. 分数提升：使用数字滚动动画

### 节奏控制
1. 开头5秒必须抓住注意力
2. 每个功能展示不超过5秒
3. 结尾CTA清晰明确

### BGM选择
- 轻快、积极、科技感
- 避免过于激昂或沉闷
- 音量适中，不压过配音

---

## 文案要点

### 必须传达的核心信息
1. 不是题库抽题，是AI生成
2. 自适应难度，个性化学习
3. 智能提示，培养思考
4. 每天15分钟，高效复习

### 避免的表述
- ❌ "海量题库"
- ❌ "名师出题"
- ❌ "直接给答案"
- ❌ "快速提分"（改为"精准提升"）

### 推荐的表述
- ✅ "AI实时生成"
- ✅ "根据薄弱点出题"
- ✅ "自适应难度"
- ✅ "智能提示引导"

## 推广渠道适配

### 微信视频号 (60秒版)
- 节奏更快
- 前3秒必须出产品名
- 结尾CTA更直接

### 抖音 (90秒版)
- 增加趣味性
- 加入"我的真实体验"叙述
- 更多数据支撑

### 小红书 (45秒版)
- 聚焦一个核心功能
- 使用"我发现了..."开头
- 强调"学习效率提升"
