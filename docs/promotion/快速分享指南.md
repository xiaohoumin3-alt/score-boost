## 🚫 避免的误区

### ❌ 不要做的事
1. 不要一天发太多条（会被屏蔽）
2. 不要在无关群发广告（会被踢）
3. 不要夸大效果（说"保证提分50分"）
4. 不要忽略用户问题（及时回复很重要）

### ✅ 应该做的事
1. 先提供价值，再提产品
2. 真实分享，不说空话
3. 及时回复评论和私信
4. 收集反馈，持续改进
