## 📋 任务生成逻辑

AI 每日任务基于以下因素生成：
1. **前一天的答题情况** - 错题优先
2. **Memory 系统记录** - 薄弱点优先
3. **知识点权重** - 高频考点优先
4. **间隔重复算法** - 需复习的知识点优先

## 💡 每日任务使用技巧

### 1. 最佳练习时间
```
🌅 早上：7:00-7:30（起床后）
  • 完成 AI 每日任务
  • 大脑清醒，记忆效果好

🌙 晚上：19:00-20:00（晚饭后）
  • 错题回顾
  • 针对性练习
  • 查看进度报告
```

### 2. 如何对待错题
```
❌ 不要：直接跳过，看答案
✅ 应该：仔细看智能提示，理解为什么错

示例：
错题：河北省的简称是？
你的答案：豫（❌）
智能提示：河北的简称是冀，河南的简称是豫
你的理解：哦，河北在北，河南在南，记住了！

下次遇到：河北省的简称？
你的答案：冀（✅）→ 知识点已掌握
```

### 3. 如何判断任务完成
```
✅ 任务完成标准：
1. 完成今日推荐题目
2. 正确率达到目标（基础65%，中等75%，优秀85%）
3. 昨天错题已理解
4. 查看了进度报告

如果没达标：
• 正确率低 → AI 会自动降低难度
• 错题没懂 → 明天继续出同类题
• 时间不够 → 至少完成每日任务（3-5题）
```
