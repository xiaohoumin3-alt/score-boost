## 📦 可部署的工具

### 1. 推广落地页 ⭐ 最重要

**文件**: `docs/promotion/deployable-website/index.html`

**功能**：
- 完整的产品介绍页面
- 倒计时显示
- 用户评价展示
- 常见问题解答
- 分享按钮
- 百度统计集成

**部署方式**：

#### 方式A: GitHub Pages（免费，5分钟）
```bash
# 1. 创建 gh-pages 分支
git checkout --orphan gh-pages
git filter-branch --subdirectory-filter docs/promotion/deployable-website -- . 
git checkout main

# 2. 推送
git push origin gh-pages

# 3. 在 GitHub 设置中启用 Pages
# Settings → Pages → Source: gh-pages branch
# 访问: https://xiaohoumin3-alt.github.io/score-boost/
```

#### 方式B: Vercel（免费，1分钟）
```bash
# 1. 安装 Vercel CLI
npm i -g vercel

# 2. 部署
cd docs/promotion/deployable-website
vercel

# 完成！会获得一个 .vercel.app 域名
```

#### 方式C: Netlify（免费，拖拽部署）
1. 访问 https://netlify.com
2. 将 `docs/promotion/deployable-website` 文件夹拖进去
3. 完成！

## 📊 数据追踪配置

### 百度统计

1. 注册百度统计：https://tongji.baidu.com/
2. 获取统计代码
3. 替换 `index.html` 中的 `YOUR_BAIDU_ID`

### UTM参数

使用分享链接生成器自动生成带追踪的链接：
```
https://yoursite.com?utm_source=wechat&utm_medium=social&utm_campaign=exam2026
```

## 🚨 快速故障排除

### 问题：GitHub Pages 404
**解决**：确保分支名称正确，设置为 `gh-pages`

### 问题：Vercel 部署失败
**解决**：确保在正确的目录运行 `vercel` 命令

### 问题：样式错乱
**解决**：清除浏览器缓存，使用隐私模式测试
