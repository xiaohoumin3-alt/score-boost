# 抖音短视频脚本 - 3条爆款模板

> 目标：15-30秒，完播率>50%，引导搜索

---

## 脚本1：痛点切入型

**时长：20秒**
**BGM：紧张节奏音乐**

```
[0-3秒] 镜头：模拟考成绩单，65分用红圈标出
画外音：生地会考只剩30天，孩子还在65分徘徊？

[3-8秒] 镜头：孩子埋头刷题，桌上堆满试卷
画外音：每天刷题到11点，分数就是上不去...

[8-15秒] 镜头：手机屏幕，AI生成题目
画外音：问题在这里！盲目刷题不如精准训练
这个AI能自动找孩子的薄弱点
每个学生题目都不一样！

[15-20秒] 镜头：小程序界面，显示"日日守护"
画外音：微信搜"日日守护"
30天精准提升，不是题库抽题！

[结尾] 字幕：前100名免费体验7天
```

## 脚本2：对比型

**时长：25秒**
**BGM：前段低沉，后段激昂**

```
[0-5秒] 分屏对比
左边：传统题库，随机抽题，孩子皱眉
右边：AI分析，精准出题，孩子点头
画外音：同样30天，为什么别人提分更快？

[5-12秒] 镜头：AI分析界面
画外音：传统刷题：会的还在做，不会的还是不会
AI精准训练：只练不会的，每道题都有价值

[12-18秒] 镜头：提分数据 65→88
画外音：我家孩子30天，从65分提到88分
关键是：每天只花15分钟

[18-25秒] 镜头：小程序搜索界面
画外音：微信搜"日日守护"
立即开始AI精准复习

[结尾] 字幕：生地会考只有一次
```

```
只练你需要练的

[12-15秒] 镜头：二维码

今天就开始

[结尾] 字幕：立即体验
```

- 关键信息用彩色强调
