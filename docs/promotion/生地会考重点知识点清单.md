## 📌 易错知识点提醒

### 生物易错点
| 知识点 | 易错点 | 提醒 |
|--------|--------|------|
| 显隐性 | AA×aa → 子代都是Aa，不是AA或aa | 记住图解 |
| 免疫 | 抗体不是抗原，抗原是外来物质 | 搞清关系 |
| 光合作用 | 光合作用需要光，呼吸作用不需要光 | 区分条件 |
| 关节 | 关节头=关节窝，关节囊=关节腔 | 区分结构 |
| 变态发育 | 蛙是完全变态，蝗虫是不完全变态 | 不要记混 |

### 地理易错点
| 知识点 | 易错点 | 提醒 |
|--------|--------|------|
| 省级行政区 | 河北=冀，河南=豫，不要搞混 | 多记简称 |
| 河流 | 黄河注入渤海，长江注入东海 | 分清注入 |
| 山脉 | 东西走向：天山-阴山；南北走向：贺兰山 | 记住走向 |
| 气候 | 季风区=夏季风影响区，不是降水多的地方 | 搞清概念 |
| 分界线 | 秦岭-淮河是800mm线，不是400mm | 记住数值 |
