# 生地会考推广策略执行方案

> 执行周期：2026年5月25日 - 6月25日
> 目标：利用生地会考契机，推广 ScoreBoost (日日守护) 小程序

---

## 📊 项目定位

### 产品名称
**日日守护** - AI驱动的生地会考复习小程序

### 核心卖点
1. **AI个性化出题** - 不是题库抽题，而是根据薄弱点实时生成
2. **自适应难度** - 自动调整匹配学生水平
3. **智能提示系统** - 不直接给答案，培养思考能力
4. **Memory系统** - 记住学习情况，生成复习计划
5. **全内容覆盖** - 7-8年级生物地理全4册

### 目标用户
- **主要**：初二学生（生地会考考生）
- **次要**：初二学生家长
- **地域**：全国实行生地会考的省市

## 📢 推广渠道与策略

### 1. 微信生态（主战场）

#### 1.1 微信公众号
**内容计划**：
- 发布频率：每周2-3篇
- 内容类型：
  - 干货文章（学习方法、知识点总结）
  - 产品介绍（AI功能说明）
  - 用户案例（真实提分故事）

**文章选题**：
1. 《生地会考30天冲刺计划》
2. 《AI如何帮你精准复习？》
3. 《从50分到85分的真实案例》
4. 《会考最容易丢分的10个知识点》

**推广方式**：
- 家长群转发
- 朋友圈分享
- 教育类公众号互推

#### 1.2 家长微信群
**目标群组**：
- 初二家长群
- 生地会考交流群
- 本地教育群

**推广话术**：
- 痛点切入："孩子复习效率低？"
- 数据说话："AI精准复习，效率提升10倍"
- 限时福利："前100名免费体验7天"

**执行计划**：
- 每天5个群
- 每群1-2条消息
- 避免硬广，先提供价值

#### 1.3 朋友圈广告
**定向投放**：
- 地域：生地会考省份
- 年龄：35-45岁（家长）
- 兴趣：教育、K12、学习辅导

**创意方向**：
- 痛点型："孩子还在盲目刷题？"
- 数据型："30天从60分到85分"
- 对比型："传统复习 vs AI复习"

### 2. 小红书（种草平台）

#### 内容策略
**账号定位**：
- 学习博主分享真实体验

**内容类型**：
1. 学习方法类
2. 产品测评类
3. 提分案例类
4. 知识点干货类

**笔记计划**：

- 覆盖不同角度
- 真实图片+数据

**话题标签**：

### 3. 短视频平台（抖音/视频号）

#### 视频内容
**类型1：功能演示**
- 展示AI出题过程

**类型2：用户故事**

**类型3：知识干货**
- 会考重点知识点
- 学习方法分享

**发布计划**：
- 视频号：每日1条
