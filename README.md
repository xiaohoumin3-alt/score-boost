# 日日守护 - AI驱动的生地会考复习小程序

> 最后30天冲刺！AI帮你精准复习，轻松拿A+

## 🎯 产品介绍

**日日守护**是一款专为初二生地会考设计的AI复习小程序，采用AI原生架构，为每个学生生成个性化题目，实现高效精准复习。

### 核心功能

| 功能 | 说明 |
|------|------|
| 🤖 **AI个性化出题** | 不是从题库抽题，而是根据学生薄弱点实时生成 |
| 📊 **自适应难度** | 连续3题正确→降难度，连续2题错误→升难度 |
| 💡 **智能提示系统** | 分级提示，不直接给答案，培养思考能力 |
| 🧠 **Memory系统** | 记住学习情况，自动生成复习计划 |
| 📚 **全内容覆盖** | 7-8年级生物地理全4册，人教版知识点 |

## 💡 技术架构

### AI原生功能

```
用户答题 → AI分析薄弱点 → 实时生成新题 → 自适应调整 → Memory记录
```

### 云函数架构

- `generateAiQuestion` - AI题目生成
- `studentMemory` - 学习记忆系统
- `generateDailyTask` - 每日任务生成
- `submitPracticeResult` - 结果提交与分析

### 数据库设计

- `users` - 用户信息
- `kp_progress` - 知识点掌握进度
- `practice_records` - 练习记录
- `daily_tasks` - 每日任务

## 🛠️ 开发相关

### 技术栈

- **前端**：微信小程序原生框架
- **后端**：腾讯云开发（云函数+云数据库）
- **AI**：Claude API（题目生成）

### 本地开发

```bash

# 运行测试
```
